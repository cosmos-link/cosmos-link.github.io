# 程序员本地博客

一个基于浏览器本地存储的程序员博客网站，无需后端服务器，数据完全存储在用户本地浏览器中。

## 功能特性

- 📝 博客撰写与编辑
- 🏷️ 标签分类管理
- 📋 博客列表查看
- 🔍 博客详情展示
- 💾 本地数据存储
- 📱 响应式设计
- ✨ 简单的Markdown支持

## 使用方法

1. 直接打开 `index.html` 文件即可使用
2. 点击"写博客"按钮创建新博客
3. 在博客列表中可以查看所有博客
4. 点击博客可以查看详情
5. 在详情页可以编辑或删除博客

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage API

## 数据存储

所有博客数据都存储在浏览器的 LocalStorage 中，键名为 `blogs`。

## 注意事项

- 数据仅保存在当前浏览器中
- 清除浏览器数据会删除所有博客
- 建议定期备份重要内容

## 浏览器兼容性

支持所有现代浏览器（Chrome、Firefox、Safari、Edge）。
